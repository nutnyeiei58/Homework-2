ความเรียงและทัศนคติ

เหตุผลที่อยากเรียน Full stack web development

เนื่องจากผมมีความสนใจในการพัฒนาซอฟต์แวร์และแอปพลิเคชันอย่างครบวงจร การเรียน Full Stack ช่วยให้ผมเข้าใจทั้งโครงสร้างหน้าบ้าน (Frontend) ที่ผู้ใช้งานมองเห็น และระบบหลังบ้าน (Backend) ที่จัดการข้อมูลและเซิร์ฟเวอร์ มันทำให้ผมสามารถสร้างสรรค์โปรเจกต์หรือไอเดียต่าง ๆ ให้เกิดขึ้นจริงได้ด้วยตัวคนเดียว และเป็นทักษะที่ตลาดแรงงานในปัจจุบันมีความต้องการสูงมากครับ

ความคาดหวังต่อวิชานี้

ผมคาดหวังว่าจะได้เรียนรู้กระบวนการพัฒนาเว็บอย่างเป็นระบบ ตั้งแต่การออกแบบ UI/UX การเขียนโค้ดเชื่อมต่อฐานข้อมูล ไปจนถึงการนำเว็บไซค์ขึ้นระบบจริง (Deployment) รวมถึงได้ฝึกฝนการแก้ปัญหาเฉพาะหน้าในการเขียนโปรแกรม และสามารถนำความรู้ที่ได้ไปต่อยอดในการทำโปรเจกต์จบหรือใช้ในการทำงานจริงในอนาคตครับ

อะไรจุดอ่อน ที่ตนเองต้องพัฒนาเกี่ยวกับทักษะ Programming และ แนวทางการพัฒนา

จุดอ่อนของผมในตอนนี้คือเรื่องของการจัดระเบียบโครงสร้างโค้ด (Clean Code) และการคิดลอจิกของอัลกอริทึมที่ซับซ้อน ซึ่งบางครั้งทำให้ใช้เวลาในการดีบั๊ก (Debug) หาข้อผิดพลาดค่อนข้างนาน แนวทางการพัฒนาของผมคือการฝึกฝนเขียนโค้ดอย่างสม่ำเสมอในทุกๆ วัน ศึกษาโค้ดจากผู้เชี่ยวชาญใน GitHub และพยายามฝึกทำโจทย์แนวพัฒนาอัลกอริทึมให้มากขึ้นเพื่อเพิ่มความชำนาญครับ
